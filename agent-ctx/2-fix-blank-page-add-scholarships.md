# Task 2: Fix Blank Page & Add Back Removed Scholarships

## Summary
Fixed the blank page issue caused by a stale Prisma Client cache and added 8 removed scholarships back to the seed file.

## Changes Made

### Issue 1: Blank Page Fix
- **Root Cause**: The Next.js Turbopack dev server cached an old Prisma Client that didn't include `priorityCourses` and `isAcceptingApplications` fields. The SQL queries generated by the cached client omitted these columns, causing the API to return incomplete scholarship data.
- **Fix**: Cleared the `.next` cache directory, regenerated the Prisma Client, and restarted the dev server. After restart, all fields are returned correctly.
- **Additional Fix**: Fixed Footer hydration mismatch by replacing `{new Date().getFullYear()}` with static `2026` in `src/components/footer.tsx`.

### Issue 2: Added Back 8 Scholarships
Added 8 scholarships to `prisma/seed.ts` under a new "GENERAL PUBLIC SCHOLARSHIPS" section:
1. UP Oblation Scholarship (University of the Philippines)
2. Benilde Academic Scholarship (De La Salle-College of Saint Benilde)
3. Mapua Financial Aid Program (Mapua University)
4. Ateneo College Scholarship Program (Ateneo de Manila University)
5. DLSU Star Scholar Program (De La Salle University)
6. UST Academic Scholarship (University of Santo Tomas)
7. CHED StuFAPs - Student Financial Assistance Programs (CHED)
8. PESFA - Private Education Student Financial Assistance (CHED)
